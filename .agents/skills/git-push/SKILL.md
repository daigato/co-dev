# Git Auto Sync スキル (`git-push`)

ユーザーから「pushして」「プッシュして」「コミットして」「変更を反映して」「Gitに上げて」などの指示があった場合、または作業区切りで同期を求められた場合は、以下の手順に従って実行してください。

## 実行手順

1. **追加とコミット**:
   変更内容（`git diff` や `git status`）を確認し、変更・新規作成されたファイルをステージングして、作業内容に応じた分かりやすいコミットメッセージを生成してコミットします。
   ```bash
   git add .
   git commit -m "chore: [具体的な作業内容]"
   ```

2. **リモートへのプッシュ**:
   現在のブランチの変更をリモートリポジトリへプッシュします。
   ```bash
   git push origin main
   ```

3. **実行完了の報告**:
   コミットメッセージとプッシュ完了の結果をユーザーに簡潔に報告します。
